# 🎨 AI Content Generator

Generate images and audio from text - completely free!

## Features
- **Text-to-Image**: SDXL-Turbo (fast, 4-step generation)
- **Text-to-Audio**: Edge TTS (8 voices, multiple languages)

## Usage
1. Enter a text prompt
2. Click Generate
3. Download your creation!
